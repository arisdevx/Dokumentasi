# API Rules

Add confirmed API-wide rules here. These rules should apply across endpoints
unless a specific endpoint says otherwise.

Useful topics include:

- authentication behavior
- required headers
- request and response formats
- error format
- pagination
- filtering and sorting
- rate limits
- idempotency
- versioning
- date, time, and timezone format
- file upload behavior
- webhook behavior
- deprecation policy
