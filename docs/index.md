# API Documentation

This documentation project is ready for your API content.

Use Markdown pages for narrative guidance and `openapi/openapi.yaml` for the API
contract. Generated endpoint references, search data, and AI-readable artifacts
are derived from those trusted sources.

## Start here

- Add your API contract to `openapi/openapi.yaml`.
- Update [Quick Start](/quick-start/) with the first successful request path.
- Update [Authentication](/authentication/) with the real authentication model.
- Update [Environment](/environment/) with approved base URLs.

## For AI agents

AI agents should start from `/llms.txt`, then use `/openapi.json` and
`/ai/integration-guide.md` before generating client code. Endpoint-specific
guidance appears after endpoints are added to OpenAPI.
