# Dokumentasi API

Dokumentasi is a lightweight API documentation starter for one API project. It
combines Markdown guides, OpenAPI endpoint references, a Try It console, and
AI-readable artifacts.

Use this documentation to understand the API contract, make your first request,
and give AI coding tools enough context to integrate the API safely.

## Start here

- Read [Quick Start](/quick-start/) to make the first request.
- Read [Authentication](/authentication/) before calling protected endpoints.
- Read [API Rules](/api-rules/) for API-wide behavior.
- Explore endpoint references from the sidebar.

## For AI agents

AI agents should start from `/llms.txt`, then use `/openapi.json` and
`/ai/integration-guide.md` before generating client code.
